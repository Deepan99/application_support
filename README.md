# Application Support Analyst - Interview Preparation Guide

This repository contains a comprehensive suite of preparation materials, cheat sheets, and technical guides designed for a Senior Application Support Analyst (L2/L3) role, specifically tailored for enterprise banking environments like BNP Paribas.

## 📚 Repository Contents

### 1. Strategy & Behavioral Preparation
* **[Textbook Definitions Cheat Sheet](textbook_definitions.md)**: Exact formal definitions for ITIL, Security, and DevOps terms (for interviewers checking boxes).
* **[Job Description Analysis](jd_analysis.md)**: A strategic breakdown mapping core support competencies to the job requirements.
* **[Interview Scenarios (STAR Method)](interview_scenarios.md)**: Real-world technical and behavioral scenarios (e.g., handling P1 outages, failed Ansible deployments) answered using the Situation-Task-Action-Result method.
* **[The Golden Rules: Do's and Don'ts](dos_and_donts.md)**: A critical guide on exactly what to say (and what to avoid saying) during an enterprise support interview.

### 2. Core Technical Guides
* **[Linux, HP-UX & Ansible Prep](tech_prep_linux_ansible.md)**: A cheat sheet for critical OS commands (including HP-UX specifics like `bdf` and `glance`), log analysis, and Ansible ad-hoc commands for troubleshooting.
* **[Ansible Practice Playbooks](ansible_practice_playbooks.md)**: Five highly practical playbooks for Application Support, including Automated Health Checks, Nginx Auto-Fix, Log Cleanup, and dynamic user management.
* **[ITIL & Support Processes](itil_support_processes.md)**: Best practices for Incident vs. Problem Management, Known Error Database (KEDB) maintenance, and major incident stakeholder communication templates.

### 3. Database & "Unknowns" Survival Guides
* **[30-Minute Survival Guide](30_min_survival_guide.md)**: A crash course on how to confidently address topics you might lack direct experience in (Databases, CI/CD, Terraform, AI), pivoting to your core strengths.
* **[SQL Basics Cheat Sheet](sql_basics_cheat_sheet.md)**: Core SQL commands (SELECT, UPDATE, JOINS) and crucial database concepts (Indexes, Transactions, COMMIT/ROLLBACK).
* **[5 Common Database Problems & Fixes](db_support_scenarios.md)**: An Application Support perspective on troubleshooting Connection Pool Exhaustion, Blocking Locks, Slow Queries, Tablespace issues, and Network Drops.
